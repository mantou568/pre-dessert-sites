# Pre-Dessert Sites 站点配置库

这是一个开源的站点适配配置文件仓库。大家群策群力，欢迎提交 Pull Requests，推动站点配置更新，人人为我，我为人人。

## 📁 项目结构

```
pre-dessert-sites/
├── site_config/                  # 站点配置文件
│   ├── commons/                  # 公共配置
│   └── sites/                    # 单个站点配置
├── site_favicon/                 # 站点图标（128×128 PNG 格式）
│   └── *.png                     # 各站点的 favicon 图标
├── tools/                        # 辅助工具脚本
```

## 📖 使用说明

本配置文件可与相关媒体管理工具配合使用，实现自动站点适配和数据获取。
